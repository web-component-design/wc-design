## 图标

- 图标用于`见名之义`的提示，通过`Icon`可以迅速了解其所代表的功能,或者对功能模块起辅助作用
- 图标采用`svg`形式，如需添加图标，请[联系我们](https://github.com/web-component-design/wc-design/issues)
